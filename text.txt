Criterion 1: Relevant to SDGs
Justification: The project directly supports SDG 4 (Quality Education) by providing students with a free, open-source tool for academic planning and time management, helping improve educational outcomes and organization.

Criterion 2: Open Source License
Justification: All source code for OpenStudy Planner is licensed under MIT License, allowing free use, modification, and distribution. Reference: https://github.com/w0nsdoof/OpenStudy/blob/main/LICENSE

Criterion 3: Clear Ownership
Justification: The project is maintained by the OpenStudy Planner development team, clearly documented in README.md with proper attribution and contact information for contributors and users.

Criterion 4: Platform Independence
Justification: OpenStudy Planner is a web application that works in any modern browser (Chrome, Firefox, Safari, Edge) and is not tied to any specific operating system or device, ensuring universal accessibility.

Criterion 5: Documentation
Justification: Comprehensive documentation is provided including installation guides, API documentation, and user guides. Available at: https://github.com/w0nsdoof/OpenStudy/blob/main/README.md and in the /docs directory.

Criterion 6: Non-Discrimination & Do No Harm
Justification: We strictly follow non-discrimination principles. Our project includes CODE_OF_CONDUCT.md prohibiting harassment, and the platform is designed to be accessible to all students regardless of background, with features supporting diverse learning needs.

Criterion 7: Privacy and Data Security
Justification: OpenStudy Planner collects minimal data (only email for login). We do not sell data to third parties and implement data encryption at rest. The privacy policy is detailed in PRIVACY.md.
